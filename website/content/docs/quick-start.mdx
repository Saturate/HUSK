title: Quick Start
description: Get YAMS running with Docker Compose in under a minute.
---

## Start the stack

```bash
git clone https://github.com/Saturate/YAMS.git
cd YAMS
docker compose up -d
```

This starts three services:

- **YAMS** — the memory server (port 3000)
- **Qdrant** — vector database for semantic search
- **Ollama** — local embedding model

The embedding model (`nomic-embed-text`) is pulled automatically on first run.

## Create your admin account

Open [http://localhost:3000/setup](http://localhost:3000/setup) and create your admin user.

## Create an API key

1. Log in at [http://localhost:3000](http://localhost:3000)
2. Go to **API Keys** → **Create Key**
3. Copy the key (`yams_...`) — you won't see it again

## Connect Claude Code

Add a `.mcp.json` to your project root (or `~/.claude/.mcp.json` for global access):

```json
{
  "mcpServers": {
    "yams": {
      "type": "http",
      "url": "http://localhost:3000/mcp",
      "headers": {
        "Authorization": "Bearer yams_your-api-key-here"
      }
    }
  }
}
```

Restart Claude Code. The MCP tools (`search`, `remember`, `list_projects`, `session_context`, etc.) are available immediately.

See [Connecting](/docs/connecting) for more options including the Claude Code plugin.
